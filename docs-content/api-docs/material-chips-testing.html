ChipHarnessFilters

A set of criteria that can be used to filter a list of chip instances.

Properties
Name Description

text: string | RegExp

Only find instances whose text matches the given value.
Deprecated

selected: boolean

Only find chip instances whose selected state matches the given value.

Name Description
