MatFormFieldControlHarness extends ComponentHarness

Base class for custom form-field control harnesses. Harnesses for custom controls with form-fields need to implement this interface.

A set of criteria shared by any class derived from MatFormFieldControlHarness, that can be used to filter a list of those components.
