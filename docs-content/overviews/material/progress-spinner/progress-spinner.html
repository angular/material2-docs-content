<mat-progress-spinner> and <mat-spinner> are a circular indicators of progress and activity.

Progress mode

The progress-spinner supports two modes, "determinate" and "indeterminate". The <mat-spinner> component is an alias for <mat-progress-spinner mode="indeterminate">.

Mode Description
determinate Standard progress indicator, fills from 0% to 100%
indeterminate Indicates that something is happening without conveying a discrete progress

The default mode is "determinate". In this mode, the progress is set via the value property, which can be a whole number between 0 and 100.

In "indeterminate" mode, the value property is ignored.

Accessibility

MatLegacyProgressSpinner implements the ARIA role="progressbar" pattern. By default, the spinner sets aria-valuemin to 0 and aria-valuemax to 100. Avoid changing these values, as this may cause incompatibility with some assistive technology.

Always provide an accessible label via aria-label or aria-labelledby for each spinner.
