API reference for Angular Material autocomplete

import {MdAutocompleteModule} from '@angular/material';

Directives

MdAutocomplete

Selector: md-autocomplete
Exported as: mdAutocomplete
Properties
NameDescription

showPanel

Whether the autocomplete panel should be visible, depending on option length.

panel

Element for the panel containing the autocomplete options.

@Input()

displayWith

Function that maps an option's control value to its display value in the trigger.

id

Unique ID to be used by autocomplete trigger's "aria-owns" property.

MdAutocompleteTrigger

Selector: input[mdAutocomplete]
Properties
NameDescription
@Input('mdAutocomplete')

autocomplete

panelOpen

panelClosingActions

A stream of actions that should close the autocomplete panel, including when an option is selected, on blur, and when TAB is pressed.

optionSelections

Stream of autocomplete option selections.

activeOption

The currently active option, coerced to MdOption type.

Methods
openPanel

Opens the autocomplete suggestion panel.

closePanel

Closes the autocomplete suggestion panel.
