getDefaultComponentOptions

Returns the default options for the @schematics/angular:component schematic which would have been specified at project initialization (ng new or ng init).

This is necessary because the Angular CLI only exposes the default values for the "--style", "--inlineStyle", "--skipTests" and "--inlineTemplate" options to the "component" schematic.
