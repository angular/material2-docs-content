progress-spinner

Module: MdProgressSpinnerModule

Directives

MdProgressSpinner

<md-progress-spinner> component.

Properties

NameDescription
@Input()

color

The color of the progress-spinner. Can be primary, accent, or warn.
@Input()

value

Value of the progress circle. It is bound to the host as the attribute aria-valuenow.
@Input()

mode

Mode of the progress circle Input must be one of the values from ProgressMode, defaults to 'determinate'. mode is bound to the host as the attribute host.

MdSpinner

<md-spinner> component. This is a component definition to be used as a convenience reference to create an indeterminate <md-progress-spinner> instance.
