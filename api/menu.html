API reference for Angular Material menu

import {MdMenuModule} from '@angular/material';

Directives

MdMenu

Selector: md-menu
Exported as: mdMenu
Properties
NameDescription
@Input('class')

classList

This method takes classes set on the host md-menu element and applies them on the menu template that displays in the overlay container. Otherwise, it's difficult to style the containing menu from outside the component.

@Input()

overlapTrigger

Whether the menu should overlap its trigger.

@Input()

xPosition

Position of the menu in the X axis.

@Input()

yPosition

Position of the menu in the Y axis.

@Output()

close

Event emitted when the menu is closed.

direction

Layout direction of the menu.

items

List of the items inside of a menu.

parentMenu

Parent menu of the current menu panel.

templateRef

Methods
focusFirstItem

Focus the first item in the menu. This method is used by the menu trigger to focus the first item when the menu is opened by the ENTER key.

hover

Stream that emits whenever the hovered menu item changes.

Returns
Observable<MdMenuItem>

setElevation

Sets the menu panel elevation.

Parameters

depth

number

Number of parent menus that come before the menu.

setPositionClasses

It's necessary to set position-based classes to ensure the menu panel animation folds out from the correct direction.

Parameters

posX?

any

posY?

any

MdMenuItem

This directive is intended to be used inside an md-menu tag. It exists mostly to set the role attribute.

Selector: [md-menu-item]
Exported as: mdMenuItem, matMenuItem
Properties
NameDescription

hover

Stream that emits when the menu item is hovered.

Methods
focus

Focuses the menu item.

MdMenuTrigger

This directive is intended to be used in conjunction with an md-menu tag. It is responsible for toggling the display of the provided menu instance.

Selector: [md-menu-trigger-for] [mdMenuTriggerFor]
Exported as: mdMenuTrigger
Properties
NameDescription
@Input('mdMenuTriggerFor')

menu

References the menu instance that the trigger is associated with.

@Output()

onMenuClose

Event emitted when the associated menu is closed.

@Output()

onMenuOpen

Event emitted when the associated menu is opened.

dir

The text direction of the containing app.

menuOpen

Whether the menu is open.

Methods
closeMenu

Closes the menu.

focus

Focuses the menu trigger.

openMenu

Opens the menu.

toggleMenu

Toggles the menu between the open and closed states.

triggersSubmenu

Whether the menu triggers a sub-menu or a top-level one.

Returns
boolean
