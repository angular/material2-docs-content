API reference for Angular Material button-toggle

import {MatButtonToggleModule} from '@angular/material/button-toggle';

Additional classes

MatButtonToggleGroupMultiple

Deprecated

MatButtonToggleChange

Change event object emitted by MatButtonToggle.

Properties
NameDescription

source: MatButtonToggle

The MatButtonToggle that emits the event.

value: any

The value assigned to the MatButtonToggle.

MatButtonToggleGroup

Exclusive selection button toggle group that behaves like a radio-button group.

Properties
NameDescription

change: EventEmitter<MatButtonToggleChange>

Event emitted when the group's value changes.

disabled: boolean

Whether the component is disabled.

multiple: boolean

Whether multiple button toggles can be selected.

name: string

name attribute for the underlying input element.

value: any

Value of the toggle group.

vertical: boolean

Whether the toggle group is vertical.

Deprecated

selected: MatButtonToggle | MatButtonToggle[] | null

Selected button toggles in the group.

MatButtonToggle

Single button inside of a toggle group.

Properties
NameDescription

ariaLabel: string

Attached to the aria-label attribute of the host element. In most cases, arial-labelledby will take precedence so this may be omitted.

ariaLabelledby: string | null

Users can specify the aria-labelledby attribute which will be forwarded to the input element

buttonToggleGroup: MatButtonToggleGroup

The parent button toggle group (exclusive selection). Optional.

change: EventEmitter<MatButtonToggleChange>

Event emitted when the group value changes.

checked: boolean

Whether the button is checked.

disableRipple: boolean

Whether ripples are disabled.

disabled: boolean

Whether the button is disabled.

id: string

The unique ID for this button toggle.

inputId: string

Unique ID for the underlying input element.

name: string

HTML's 'name' attribute used to group radios for unique selection.

value: any

MatButtonToggleGroup reads this to assign its own value.
